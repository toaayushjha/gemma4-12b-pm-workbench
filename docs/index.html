It really read this

One vision task: "read this KPI dashboard and tell me the most concerning number." Gemma reported all six KPIs with exact values and trends, correctly singled out the Activation Rate (41%, −2.0%), and recommended an onboarding-funnel audit.

Where each one wins

✅ Local Gemma 4 12B

  • Summarizing & writing — ship-ready drafts, with citations.
  • Audio — native transcription + diarization, no Whisper, no cloud.
  • Vision / OCR — exact dashboard reads, chart→table, scanned docs.
  • $0, private, offline — for sensitive work, it never leaves the laptop.

🔵 Claude Opus 4.8

  • Speed — replies feel instant vs. 30–90s locally.
